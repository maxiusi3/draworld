# 🔧 开通阿里云通义万相2.2付费模式指南

## 📋 当前状态检查

如果您看到以下错误，说明需要开通付费模式：

```json
{
  "code": "AllocationQuota.FreeTierOnly",
  "message": "The free tier of the model has been exhausted. If you wish to continue access the model on a paid basis, please disable the 'use free tier only' mode in the management console."
}
```

## 🚀 开通步骤（详细图文）

### 步骤1：登录阿里云百炼控制台

1. 打开浏览器，访问：https://bailian.console.aliyun.com/
2. 使用您的阿里云账号登录
3. 如果没有账号，请先注册并完成实名认证

### 步骤2：进入模型管理

1. 在左侧导航栏中，点击 **"模型广场"**
2. 在搜索框中输入 **"通义万相"**
3. 找到 **"万相2.2图生视频"** 模型
4. 点击进入模型详情页

### 步骤3：管理模型服务

1. 在模型详情页，点击 **"管理"** 按钮
2. 或者直接访问：https://bailian.console.aliyun.com/model-market
3. 找到已开通的"万相2.2图生视频"服务

### 步骤4：关闭免费模式限制

1. 在服务管理页面，找到 **"服务配置"** 或 **"计费设置"**
2. 找到 **"仅使用免费额度"** 选项
3. **取消勾选** 或 **关闭** 此选项
4. 确认开通 **"按量付费"** 模式

### 步骤5：确认设置

1. 确认页面显示 **"按量付费"** 状态
2. 查看费用说明：
   - 480P：0.14元/秒
   - 1080P：0.70元/秒
3. 点击 **"确认"** 完成设置

## 🔍 常见问题

### Q1: 找不到"仅使用免费额度"选项？
**A**: 可能在不同位置，尝试查找：
- 服务配置页面
- 计费设置页面
- 模型管理页面的设置选项

### Q2: 开通后仍然报错？
**A**: 请等待5-10分钟让设置生效，然后重试

### Q3: 担心费用过高？
**A**: 可以设置：
- 费用预警：在阿里云控制台设置月度预算
- 使用限制：控制API调用频率
- 监控使用：定期查看费用统计

### Q4: 如何查看使用统计？
**A**: 访问：https://bailian.console.aliyun.com/model-telemetry
