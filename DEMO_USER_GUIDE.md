# 🎮 Whimsy Brush 演示用户指南

## 🔑 测试账户

演示环境支持多个测试账户，每个账户都有不同的初始积分和权限：

### 主要测试账户

| Token | 用户名 | 初始积分 | 用途 |
|-------|--------|----------|------|
| `demo-token` | 演示用户 | 200积分 | 主要演示账户，功能齐全 |
| `new-user-token` | 新用户 | 50积分 | 模拟新注册用户 |
| `admin-token` | 管理员 | 1000积分 | 管理员功能测试 |
| `test-user-1-token` | 测试用户1 | 150积分 | 社交功能测试 |
| `test-user-2-token` | 测试用户2 | 150积分 | 社交功能测试 |

## 🎯 功能测试指南

### 1. 💰 积分系统测试

#### 查看积分余额
- 访问 `/credits` 页面
- 或使用不同token调用API: `GET /api/credits?action=balance`

#### 每日签到
- 在积分页面点击"每日签到"
- 每个账户每天可获得10积分
- 重复签到会提示"今日已签到"

#### 积分购买
- 访问 `/credits` 页面
- 选择积分套餐（基础/高级/专业）
- 点击购买，系统会自动创建订单
- 等待3秒模拟支付完成
- 积分会自动添加到账户

### 2. 🎬 视频生成测试

#### 积分要求
- **每次生成消费**: 30积分（已优化）
- **推荐账户**: `demo-token`（200积分）或购买积分后的账户

#### 生成流程
1. 访问主页 `/`
2. 上传图片或使用示例图片
3. 输入提示词
4. 点击生成视频
5. 系统会自动扣除30积分
6. 可查看生成进度和状态

#### 积分不足处理
- 如果积分不足，系统会提示需要的积分数量
- 可以通过购买积分或每日签到获取更多积分

### 3. 🎨 社区功能测试

#### 浏览作品
- 访问 `/gallery` 页面
- 现在显示5个精美的示例作品
- 每个作品都有详细信息和创作者信息

#### 社交互动奖励
- **点赞作品**: 获得2积分
- **评论作品**: 获得5积分
- **每日限制**: 防止刷分，合理限制奖励频率

#### 多用户测试
使用不同token可以模拟多个用户：
```javascript
// 在浏览器控制台中切换用户
localStorage.setItem('auth_token', 'new-user-token');
location.reload();
```

### 4. 👥 多用户社交测试

#### 切换用户方法

**方法1: 浏览器控制台**
```javascript
// 切换到新用户
localStorage.setItem('auth_token', 'new-user-token');
location.reload();

// 切换到测试用户1
localStorage.setItem('auth_token', 'test-user-1-token');
location.reload();

// 切换回演示用户
localStorage.setItem('auth_token', 'demo-token');
location.reload();
```

**方法2: 隐私模式**
- 在不同的隐私窗口中使用不同token
- 可以同时登录多个账户

#### 社交功能验证
1. 用户A点赞用户B的作品
2. 用户A获得点赞奖励积分
3. 用户B的作品点赞数增加
4. 评论功能同样支持跨用户互动

### 5. 🛡️ 管理功能测试

#### 管理员权限
- 使用 `admin-token` 访问管理功能
- 访问 `/admin/moderation` 审核后台
- 访问 `/admin/payment-monitor` 支付监控
