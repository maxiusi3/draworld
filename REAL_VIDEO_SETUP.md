# 🎬 真实视频生成功能配置指南

本指南将帮助您配置通义万相2.2 API，启用真实的视频生成功能，替换演示模式。

## 📋 功能概述

- **演示模式**：使用模拟数据，返回示例视频
- **真实模式**：调用通义万相2.2 i2v fast模型，生成真实视频

## 🚀 快速开始

### 1. 自动配置（推荐）

运行配置脚本：

```bash
npm run setup:video
```

脚本将引导您完成以下步骤：
- 获取API密钥指南
- 配置环境变量
- 测试API连接
- 验证功能

### 2. 手动配置

#### 步骤1：获取API密钥

1. 访问 [阿里云百炼平台](https://bailian.console.aliyun.com/)
2. 登录您的阿里云账号
3. 进入"API-KEY管理"页面
4. 创建新的API-KEY或使用现有的
5. 复制API密钥（格式：`sk-xxxxxxxxxxxxxxxxx`）

#### 步骤2：配置环境变量

创建或编辑 `.env` 文件：

```bash
# 通义万相API配置
DASHSCOPE_API_KEY=sk-your-actual-api-key-here
```

#### 步骤3：测试配置

运行测试脚本：

```bash
npm run test:video
```

## 🔧 技术实现

### API切换逻辑

系统会自动检测是否配置了API密钥：

```javascript
const apiKey = process.env.DASHSCOPE_API_KEY;

if (!apiKey) {
  // 演示模式：返回模拟数据
  return mockVideoGeneration();
} else {
  // 真实模式：调用通义万相API
  return realVideoGeneration(apiKey);
}
```

### 支持的功能

- ✅ 图生视频（Image-to-Video）
- ✅ 使用 `wan2.2-i2v-flash` 模型
- ✅ 480P分辨率（成本优化）
- ✅ 5秒视频时长
- ✅ 自动提示词扩展
- ✅ 无水印输出
- ✅ 异步任务处理
- ✅ 实时状态查询

### API端点

- **创建任务**：`POST /api/video/start`
- **查询状态**：`GET /api/video/status?taskId=xxx`

## 💰 费用说明

- **分辨率**：480P（降低成本）

## 🛠️ 故障排除

### 常见问题

#### 1. API密钥无效
```
❌ API密钥验证失败: API密钥无效或权限不足
```

**解决方案**：
- 检查API密钥格式是否正确
- 确认API密钥权限包含视频生成
- 验证账户状态是否正常

#### 2. 余额不足
```
❌ 创建视频任务失败: 账户余额不足
```

**解决方案**：
- 登录阿里云控制台充值
- 检查账户余额和消费限制

#### 3. 图片URL无法访问
```
❌ 创建视频任务失败: 图片URL无法访问
```

**解决方案**：
- 确保图片URL可以公开访问
- 检查图片格式（支持JPG、PNG）
- 验证图片大小（建议小于10MB）

#### 4. 任务超时
```
⚠️ 视频生成超时，请稍后查询
```

**解决方案**：
- 视频生成通常需要30-60秒
- 可以稍后手动查询任务状态
- 检查网络连接稳定性

## 📊 监控和日志

### 日志级别

- `[VIDEO START API]`：视频任务创建
- `[VIDEO STATUS API]`：任务状态查询
- `[TONGYI API]`：通义万相API调用

### 状态码说明

- `pending`：任务已创建，等待处理
- `processing`：视频生成中
- `completed`：生成完成
- `failed`：生成失败
